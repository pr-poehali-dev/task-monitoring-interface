# Локальный сервер

Позволяет запускать приложение полностью офлайн с локальной базой данных SQLite.

## Требования

- [Node.js](https://nodejs.org) версия 18 или выше

## Первый запуск

1. Открыть папку `localserver` в терминале/командной строке
2. Установить зависимости (один раз):
   ```
   npm install
   ```
3. Запустить сервер:
   ```
   npm start
   ```
4. Открыть приложение в браузере: **http://localhost:5173**

> База данных `korpus.db` создаётся автоматически в папке `localserver`.
> Все данные хранятся в этом файле — делайте его резервные копии.

## Ежедневный запуск

Просто запускайте `npm start` в папке `localserver` перед открытием приложения.

## Остановка

Нажмите `Ctrl+C` в окне терминала.
